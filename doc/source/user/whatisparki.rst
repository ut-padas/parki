What is ParkiPy?
****************

The (Par)ticle (k)ernel (i)nteractions library for python (Py) leverages Kokkos [1]_ through it's python framework PyKokkos [2]_ to performance portable parallel APIs for scientific computing. Our APIs are simple to call and work seamlessly with the `numpy <https://numpy.org/>`_ array framework on CPUs and the `cupy <https://cupy.dev/>`_ array framework for GPUs.
